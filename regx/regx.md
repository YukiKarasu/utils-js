1. 查找
2. 替换

## 创建方式

1. 字面量创建

```javascript
let str = 'asdf41435a2s1df354'
let reg = /\d+/g;
const res = str.match(reg);
```

2. 构造函数创建

```javascript
let str = 'asdf41435a2s1df354'
let reg = new RegExp("\\d+", "g");
const res = str.match(reg);
```

## 相关概念

### 元字符

> 有特殊函数非字母字符

1. 字符相关
  * /w：数字、字母、下划线
  * /W: 和/w相反
  * /d: 数字
  * /D: 非数字
  * /s: 空格
  * /S: 非空格
  * .: 非(\n,\r,\u2028,\u2029)
2. 数量相关
  * +: {1, }
  * ?: {0, 1}
  * *: {0, }
3. 位置相关
  * ^: 字符开始位置
  * $: 字符结束位置
  * /b: 边界符
  * /B: 非边界符
4. 括号相关()[]{}
  * (): 分组,提取值
  * []: 分组
  * {}: 分组
  * /n: 该n为数字,内容为匹配到的分组的字符

5. 匹配模式
  * g: 全局
  * i: 忽略大小写
  * s: 让.匹配换行
  * u: unicode编码,配合/u{}使用
  * y: 粘性模式,匹配必须是连续的
  * m: 多行模式

6. ?<name>: 命名分组

7. ?=: 正向肯定零宽断言

8. ?!: 负向否定零宽断言
